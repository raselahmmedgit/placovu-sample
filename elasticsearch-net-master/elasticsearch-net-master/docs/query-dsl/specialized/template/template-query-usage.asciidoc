=== Template Query Usage

==== Fluent DSL example

[source,csharp]
----
q
.Template(sn => sn
    .Name("named_query")
    .Boost(1.1)
    .Inline(_templateString)
    .Params(p=>p.Add("query_string", "all about search"))
)
----

==== Object Initializer syntax example

[source,csharp]
----
new TemplateQuery
{
    Name = "named_query",
    Boost = 1.1,
    Inline = _templateString,
    Params = new Dictionary<string, object>
    {
        { "query_string", "all about search" }
    }
}
----
