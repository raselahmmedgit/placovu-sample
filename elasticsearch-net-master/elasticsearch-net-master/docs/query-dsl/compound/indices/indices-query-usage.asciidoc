=== Indices Query Usage

==== Fluent DSL example

[source,csharp]
----
q
.Indices(c => c
    .Name("named_query")
    .Boost(1.1)
    .Indices(Index<Project>())
    .Query(qq => qq.MatchAll())
    .NoMatchQuery(qq => qq.MatchAll(m => m.Name("no_match")))
)
----

==== Object Initializer syntax example

[source,csharp]
----
new IndicesQuery()
{
    Name = "named_query",
    Boost = 1.1,
    Indices = Index<Project>(),
    Query = new MatchAllQuery(),
    NoMatchQuery = new MatchAllQuery { Name ="no_match" }

}
----
