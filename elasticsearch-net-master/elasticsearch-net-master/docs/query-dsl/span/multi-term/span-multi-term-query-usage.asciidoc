=== Span Multi Term Query Usage

==== Fluent DSL example

[source,csharp]
----
q
.SpanMultiTerm(c => c
    .Name("named_query")
    .Boost(1.1)
    .Match(sq=>sq
        .Prefix(pr=>pr.Field(p=>p.Name).Value("pre-*"))
    )
)
----

==== Object Initializer syntax example

[source,csharp]
----
new SpanMultiTermQuery
{
    Name = "named_query",
    Boost = 1.1,
    Match = new PrefixQuery { Field = "name", Value = "pre-*" }
}
----
